## 🌟 Overview

A complete offline learning web application for **Tarjamat-ul-Quran (Class 9)** featuring quizzes, flashcards, AI assistant, and advanced progress tracking.

---

## 🚀 Features

* ✅ MCQ Quiz System (224 Questions)
* 🎯 Random Exam Generator
* ⚡ Speed Quiz Mode
* 📊 Progress Dashboard (Charts & Analytics)
* 🧠 AI Question Assistant (Offline)
* 📚 Flashcard System (88 Vocabulary Words)
* 🔁 Mistake Practice Mode
* 🔎 Real-time Question Search
* 🔖 Bookmark System
* 🎮 Matching Game
* 🎨 25 Themes (Light, Dark, Islamic, Colorful)
* 🏆 25 Achievement Badges
* 📱 Fully Responsive + Installable PWA

---



## 📦 Installation

1. Download the repository
2. Open:

```
tarjamat_quiz_v2.html
```

3. Works **100% Offline**

---

## 📱 PWA Installation

* Open in Chrome (Mobile/Desktop)
* Click **Install App**
* Use like a native app
